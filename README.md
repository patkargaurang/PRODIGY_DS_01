# PRODIGY_DS_01
## Objective
   My first internship task as Data Science Intern at Prodigy InfoTech focused on basic data visualization using Python.

## Overview
  I prepared the dataset by handling the missing values and creating visualizations including Histograms, Kernel Density Estimation, Stacked Bar Charts,
  Strip Plots and Heatmaps. This task significantly enhanced my skills in Data Analysis and visualization using :
  * Pandas
  * Matplotlib
  * Seaborn

## Conclusion 
   In this project, we visualized a demographic dataset to explore various distributions and relationships. Using histograms, bar charts, pie charts, strip plots, and heatmaps, we uncovered insights about the data's structure and trends. Key findings include diverse distributions across multiple categories and significant variations in different groups' attributes. A multi-variable bubble plot highlighted complex interactions within the data. These visualizations offer a comprehensive understanding of the dataset, revealing important socio-economic patterns and trends. Future analyses could incorporate additional variables and advanced statistical methods for deeper insights.
